# Cursor_Tcinexus__Website

Modern starter website for TCI Nexus International Ltd.

## Run locally

No build tools are required. Open `index.html` directly in your browser.

Optional local server (recommended):

```powershell
cd C:\Cursor_Schwab\Cursor_Tcinexus__Website
python -m http.server 8080
```

Then open: [http://localhost:8080](http://localhost:8080)

## What is included

- Responsive one-page layout
- Sticky navigation with mobile menu
- Photo-rich hero, service cards, and gallery strip
- Services, About, and Contact sections
- Clean modern styling with easy-to-edit CSS variables
